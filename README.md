# Yh remaster

The original Yesterhaven map, remastered.

## How to build

- Provide environment variables:
  - %DungeonSiege% - path to DS installation
  - %TankCreator% - path to TC installation
  - %GasPy% - path to GasPy repo
- Run the build.bat in the repo root
